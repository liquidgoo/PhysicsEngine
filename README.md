# Features

## Discrete collision detection
### Geometric primitives:
- Box
- Sphere
- Plane

Works with complex shapes by combining multiple primitives with one RigidBody

## Collision handling
- Velocity adjustment
- Interpenetration resolution
- Friction

## Joints
Suppots constraing point of one body to point of another body or fixed point in space
